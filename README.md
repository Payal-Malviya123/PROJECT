# Expense Tracker Web App

A simple and clean **Expense Tracker** built using **HTML, CSS, and JavaScript**.  
This project allows users to add their daily expenses, categorize them, view total spending, and delete entries easily.

# Features

-  Add new expenses with:
  - Title  
  - Amount  
  - Category  
  - Date
-  Automatically displays expenses in a styled list
-  Shows total amount spent
-  Delete any expense instantly
-  Beautiful modern UI with gradient background & card-style layout
-  Smooth hover animations

# Tech Stack

| Technology | Purpose |
|-----------|----------|
| **HTML**  | Structure of the web app |
| **CSS**   | Styling, layout, gradients, cards, animations |
| **JavaScript** | App logic, total calculation |

## ✨ How It Works

1. User enters **expense title, amount, category, and date**.
2. Click **Add Expense**.
3. Expense appears in the list below in a card-style box.
4. Total amount updates instantly.
5. User can delete any expense → total also updates.

# Setup Instructions

You can run this project in 2 ways:

# Direct Run   
Just download the files and open `index.html` in your browser.

# VS Code Live Server
- Open folder in VS Code  
- Install Live Server extension  
- Right click → **Open with Live Server**
